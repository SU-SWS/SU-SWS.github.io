Jumpstart Academic 4.3

Projected release date: Spring 2015

Primary focus: Carousel

The primary focus for this release will be the Carousel module. The work will include adding help documentation into the feature and onto the User Guide, and integrating Carousel into Jumpstart Academic by default.

Planned bug fixes:

  • Exclude blocks from showing up on admin pages in "site-wide"
  • News: Move External link field to Add/Edit External Article Information grouping
  • Change default sort on Manage Events VBO to Updated Date
  • Link news image to content in views

Additional new features:

  • Create One Page Service Sheet for Jumpstart Academic
  • Create a department member role a la Sociology
  • Events: add new Map field form events.stanford to importer and Stanford Event CT
